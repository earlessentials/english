# English Playground

A responsive, dependency-free English-learning playground built for Indonesians.

**Live site:** https://earlessentials.github.io/english/

## Run locally

Serve the folder with any static server. For example:

```sh
python3 -m http.server 43128
```

Then open `http://localhost:43128`.

## Included in this prototype

- Responsive landing experience and mobile navigation
- IndexedDB content/progress database with offline persistence
- Real calendar-based current/best streak tracking
- Spaced-repetition vocabulary review scheduling
- Six database-backed grammar lessons and six speaking scenarios
- Sixteen playable game records with validated algorithm engines
- Five-round sentence challenge increasing from 3 to 9 words
- Thirty-four database-backed learning-path lessons across Greetings, Family, and Food
- Seventeen-form tense learning center with positive, negative, and question examples
- Timed 50-question tense capability test scored out of 100
- Advanced Professional English center with a timed 20-scenario assessment and embedded video lesson
- Custom black-cat-and-pearls mascot identity
- Searchable module/lesson/game launcher
- Theme and interface-language controls
- Local guest progress using IndexedDB
- Installable PWA manifest and offline asset cache
- Reduced-motion and keyboard-focus accessibility support

Optional cloud accounts, AI writing feedback, community services, and the admin CMS require backend services in a future release. The learning content, games, browser audio, speech recognition, local progress, streaks, and assessments work without an account.
